# Sumber Logo Pembayaran CAKSTORE

Logo pembayaran di folder `src/assets/payments-official/` diambil dari sumber resmi/brand guideline dan file publik berikut:

- QRIS: Wikimedia Commons, sumber Bank Indonesia — https://commons.wikimedia.org/wiki/File:QRIS_logo.svg
- DANA: Wikimedia Commons, sumber dana.id — https://commons.wikimedia.org/wiki/File:Logo_dana_blue.svg
- GoPay: Wikimedia Commons — https://commons.wikimedia.org/wiki/File:Gopay_logo.svg
- OVO: Wikimedia Commons, sumber ovo.id — https://commons.wikimedia.org/wiki/File:Logo_ovo_purple.svg
- ShopeePay: website resmi ShopeePay — https://shopeepay.co.id/ dan brand guideline — https://product.shopeepay.com/products/brand-guidelines/
- BRI: Wikimedia Commons — https://commons.wikimedia.org/wiki/File:BANK_BRI_logo.svg
- BCA: Wikimedia Commons — https://commons.wikimedia.org/wiki/File:Bank_Central_Asia.svg

Folder `src/assets/payments/` tetap disertakan sebagai fallback lokal sederhana jika browser gagal merender salah satu file logo resmi.

Catatan: semua logo dan merek dagang tetap milik pemilik brand masing-masing. CAKSTORE tidak berafiliasi dengan brand tersebut.
