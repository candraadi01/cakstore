## Cara menjalankan

```bash
npm install
npm run dev
```

Buka:

```txt
http://localhost:5173
```

## Cara deploy/update ke GitHub dan Vercel

```bash
git add .
git commit -m "Update branding CAKSTORE"
git push
```

Jika repository sudah terhubung ke Vercel, deploy akan berjalan otomatis.
