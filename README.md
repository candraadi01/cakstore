# CAKSTORE Landing Page

CAKSTORE adalah landing page toko layanan digital premium seperti Netflix, Vidio, Viu, Prime Video, Canva, Spotify, dan layanan digital lainnya.

## Live Demo

Website dapat diakses melalui link berikut:

🔗 [https://cakstore.vercel.app/](https://cakstore.vercel.app/)

## Tech Stack

- React
- Vite
- Tailwind CSS
- Vercel

## Fitur

- Tampilan mobile-first
- Tombol order langsung ke WhatsApp
- Daftar produk premium
- Metode pembayaran
- Desain responsif untuk HP dan laptop
